# Social Media API
RESTful API для социальной медиа платформы, 

## Фунционал
* Пользователи могут регистрироваться
* Входить в систему 
* Создавать посты
* Переписываться
* Подписываться на других пользователей и добавлять в друзья 
* Получать ленту постов от друзей

## Технологий
* Java 17
* Spring Boot 3.0
* Spring Security
* PostgreSQL
* JSON Web Tokens (JWT)
* BCrypt
* Maven
* UI API Swagger-ui
* H2(для тестов)
